# proc Product Roadmap

## Product Positioning

`proc`는 취업 준비생이 프로젝트 주제를 고르고, 실무처럼 기획하고, 결과를 포트폴리오 언어로 정리할 수 있게 돕는 웹 서비스다.

## Core User Flow

1. 희망 직무와 프로젝트 기간을 선택한다.
2. 키워드를 고르고 프로젝트 추천을 받는다.
3. 추천 프로젝트를 선택해 브레인스토밍과 기획 기록을 시작한다.
4. 구현 로그와 회고를 남긴다.
5. 이력서 문장, README 초안, 예상 면접 질문으로 정리한다.

## Delivery Priorities

### 1. Recommendation

- 직무, 기간, 도메인, 난이도 기반 추천 점수
- 추천 이유와 예상 산출물 제공

### 2. Project Workspace

- 프로젝트별 기록 저장
- 브레인스토밍, 기획, 진행, 회고 템플릿

### 3. Hiring Outputs

- 포트폴리오 요약 문장
- README 초안
- 예상 면접 질문

### 4. Team Collaboration

- 로그인
- 팀원 초대
- 프로젝트 히스토리 관리
